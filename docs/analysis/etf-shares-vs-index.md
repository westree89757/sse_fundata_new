## 阶段解读框架

| 阶段 | 份额趋势 | 指数表现 | 含义 |
|------|---------|---------|------|
| 🟢 机构增持 | 份额持续上升 | 指数横盘/下跌 | 大资金在抄底，关注入场 |
| 🟡 中性观望 | 份额小幅波动 | 指数正常波动 | 市场无明显方向，持有等待 |
| 🔴 获利赎回 | 份额持续下降 | 指数上涨 | 机构获利了结，趋势完好但别追高 |

## 正确操作框架

```
份额信号触发后:

🟢 机构增持阶段:
  ├─ 确认: 连续 2 周份额上升 + 指数横盘/下跌
  ├─ 动作: 可分批建仓, 这是中期入场窗口
  └─ 注意: 规模越大 ETF 信号越可靠 (首选 510300/510310/510330)

🟡 中性阶段:
  ├─ 确认: 份额在正常区间波动
  ├─ 动作: 继续持仓, 不做调整
  └─ 注意: 中性可能持续数周, 耐心等待信号

🔴 获利赎回阶段:
  ├─ 阶段 < 2 周: 继续持有, 趋势完好, 不要被动下车
  ├─ 阶段 2-4 周: 持仓不变, 关注是否转入中性
  ├─ 阶段 > 4 周 (罕见): 开始减仓, 趋势可能衰竭
  └─ 规则: 获利赎回不是卖出信号, 是"享受上涨但别追高"的提醒

⚪ 连续 5+ 天极端信号:
  ├─ 连续增持 > 5 天 → 强买入信号, 历史上正收益概率 60-100%
  ├─ 连续赎回 > 5 天 → 检查趋势是否衰竭, 如指数仍在涨则持有
  └─ 这是最可靠的短期信号, 但出现频率低 (年均 3-5 次)
```

## 关键原则

1. **份额是滞后验证指标，不是领先预测指标** — 确认"处于什么阶段"，不预测"明天涨跌"
2. **牛市中的获利赎回是正常现象** — 机构落袋为安 ≠ 趋势反转
3. **规模越大的 ETF 信号越可靠** — 华泰柏瑞/易方达/华夏三家对机构行为最敏感
4. **负相关的经济逻辑** — 上涨时赎回(获利了结)，下跌时申购(越跌越买)，ETF 申赎机制的自然结果
5. **滞后约 1 个月** — 投资者行为存在惯性，份额变化滞后指数约 20-30 个交易日
6. **广发300ETF 近乎完美负相关(r=-0.96)** — 套利资金活跃，申赎价差套利
